# Week 1 java oop
This is my week 1 java oop practice project.

This project is a simple "vehicle system" where i practiced how a child class can extends a parent class and its own behavior.

# project structure

This project contains three main classes:
- vehicle class(parent class)
- car class(child class that extends vehicle)
- main (runs the program and tests everything)

# What the project does

The vehicle class contains basic properties like:
- brand
- year
The car class extends the vehicle class and adds extra features like:
- number of doors
- overrides the method
The main class creates objects and runs the program to test everything.
